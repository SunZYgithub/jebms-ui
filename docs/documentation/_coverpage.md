# jebms admin system <small>1.0.0</small>

> 一个基于vue做前端，后端用java提供API的前后端分离的SPA后台管理功能。

- 基础UI基于buefy(bulma)，前端UI完全响应式的，一次编写，3端可用
- 完全实现了Oracle EBS的基础功能配置架构：功能-->菜单-->职责-->用户
- 用vue封装了一个强大的灵活的表格编辑组件(table-editor)
